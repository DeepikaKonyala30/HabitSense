# 🚀 StreakMate  
### A Social Habit Tracking Platform Built for Consistency & Community
**Live Demo:** https://streakmate.vercel.app

StreakMate began with a simple observation:  
Habit building isn’t just a grind. It’s emotional, social, and deeply human.

Most habit trackers treat consistency like a cold math problem. Miss a day? Reset.  
Feeling low? Irrelevant.  
Want support? Not part of the system.

StreakMate flips that script.

It builds habits the way people actually work — with emotions, communities, and a little forgiveness when life happens.

---

## 🌟 Why StreakMate Exists

Traditional habit apps:
- Punish missed days aggressively   
- Focus only on solo progress  

StreakMate aims for something closer to real life:
- Smarter streak logic  
- Community motivation through circles  

Consistency becomes more than counters. It becomes connection.

---

## ✨ Core Features

### 🧩 Habit & Streak Engine  
- Daily habit tracking  
- Automatic streak detection  
- Smart missed-day handling  
- Controlled streak restoration  
- Cron-powered background jobs  
- Logic built for humans, not just checkboxes  

### 🤝 Circles (Social Accountability)  
- Join multiple circles  
- Request + approval flow  
- Community posts and updates  
- Shared motivation and progress  

### 🌍 Location-Based Discovery  
- Map-based circle visualization  
- Local and global discovery  
- Place-aware community building  

### 🔐 Authentication & Security  
- Middleware-protected routes  
- Token-based user validation  
- Secure frontend access  
- Clean modular API structure
